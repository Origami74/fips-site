# Diagrams

This directory contains SVG diagrams for the FIPS website. These should be copied/adapted from `docs/design/diagrams/`:

- `fips-node-architecture.svg` - Hero section diagram showing the full protocol stack
- `fips-mesh-topology.svg` - Mesh network topology with spanning tree
- `fips-protocol-stack.svg` - Layered protocol stack diagram
- `fips-architecture-overview.svg` - Packet flow across heterogeneous transports
- `fips-identity-derivation.svg` - Keypair to identity derivation

The existing SVGs in `docs/design/diagrams/` are well-crafted with consistent styling. Copy them here and adapt as needed for web use.
